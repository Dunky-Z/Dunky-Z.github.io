Hexo 和 GitHub 搭建博客以及更换电脑同步博客

7月 20 2021 1 分钟 读完 (约 174 字)

只要有source文件夹下所有源文件就可以重新部署，按照正常的搭建 Hexo 环境开始搭建，搭建好以后将source文件夹替换即可，需要应用主题就下载主题然后替换。

注意：

  • 主题更换需要更改_config_yml文件
  • _config_yml文件中的部署配置，branch:master就是每次hexo d操作推送的分支。而在命令行每次git push推送的分支是设置的默认分支hexo
